Question API

Name Description Parameters Return Value Usage

QuizBox

Given the parameter element, the function will return a JS object that represents a quiz.

Element: is a DOM element that the quiz will appear in the browser.

 0
  • Example:

    • Const elementToAddQuizTo = document.querySelector('body');

    • Const newQuiz = QuizBox(elementToAddQuizTo);

    • NOTE: the quiz will appear in the body element.

createMultipleChoiceOne

Given the parameters, question, options and answer, the function will create a multiple-choice question and add it to the quiz. Users can answer the question by selecting one of the options as their answer to the question.
  • question: is string that represents the question.

  • options:: is an array of strings where each string is an answer option to the question.

  • answer: is a string that is contained in the “options” array. This string represents the answer to the question.
 0
  • NOTE: If a developer wants to create this type of multiple-choice question, they must first create a QuizBox object.

  • Example:

    • Const elementToAddQuizTo = document.querySelector('body');

    • Const newQuiz = QuizBox(elementToAddQuizTo);

    • newQuiz.createMultipleChoiceOne (“What is 2 + 2”, [“1”, “2”, “3”, “4”], “4”);

createMultipleChoiceMany

Given the parameters, question, options and answers, the function will create a multiple-choice question and add it to the quiz. Users can answer the question by selecting multiple options as their answer to the question.
  • question: is string that represents the question.

  • options: is an array of strings where each string is an answer option to the question.

  • answer: is an array of strings that are contained in the “options” array. This array of strings represents the answers to the question.
 0
  • NOTE: If a developer wants to create this type of multiple-choice question, they must first create a QuizBox object.

  • Example:

    • Const elementToAddQuizTo = document.querySelector('body');

    • Const newQuiz = QuizBox(elementToAddQuizTo);

    • newQuiz.createMultipleChoiceMany(“Which of the following cities are located in Ontario, Canada?”, [“Hamilton”, “Toronto”, “Mississauga”, “New York”], [“Hamilton”, “Toronto”, “Mississauga”]);

createTrueOrFalseQuestion

Given the parameters, statements and answers, the function will create a true or false question and add it to the quiz. Users can answer the question by dragging tiles (each tile represents a statement) into the red box (statements in this box are considered false) or into green box (statements in this box are considered true).
  • statements: is string that represents the question.

  • answer: is an array of Boolean values. The value at an index indicates the validity of the statement at the same index in the array “statements”. For example, if the value at index 0 in the array “answers” is true then the statement at index 0 in the array “statements” is true.
 0
  • NOTE: If a developer wants to create this type of multiple-choice question, they must first create a QuizBox object.

  • Example:

    • Const elementToAddQuizTo = document.querySelector('body');

    • Const newQuiz = QuizBox(elementToAddQuizTo);

    • newQuiz.createMultipleChoiceMany([“URL stands for Uniform Resource Locator”, “HTTP stands for HyperText Translation Protocol”], [true, false]);

createSequenceQuestion

Given the parameters, events and answer, the function will create a sequence question and add it to the quiz. Users can answer the question by dragging tiles (each tile represents an event) into the blue box chronologically.
  • Events: is an array of strings where each string is an event.

  • answer: is an array of strings that are contained in the array “events”. Note that the event at index 0 is the most recent event and the event at length of “answer” - 1 is the oldest event.
 0
  • NOTE: If a developer wants to create this type of multiple-choice question, they must first create a QuizBox object.

  • Example:

    • Const elementToAddQuizTo = document.querySelector('body');

    • Const newQuiz = QuizBox(elementToAddQuizTo);

    • newQuiz.createSequenceQuestion ([“World War 2”, “World War 1”, “COVID-19”], [“COVID-19”, “World War 1”, “World War 2”]);
