Getting started:

Step 1:

To start using this library, include the script tag, <script defer type="text/javascript" src='js/QuizBox.js'></script>, and the link tag, <link rel="stylesheet" type="text/css" href="QuizBox.css"> in the HTML file. Moreover, create a JS file and include it in the same HTML file (i.e. include the script tag, <script defer type=" text/javascript" src='Path to the JS file'></script>) that contains the script tag and the link tag.


NOTE: The path for QuizBox.js and QuizBox.css could differ depending on where QuizBox.js and QuizBox.css is placed with repect to the HTML file.

Before moving on to the next step, please ensure that the following tags are included in the HTML file:

  • <script defer type="text/javascript" src='js/QuizBox.js'></script>
  • <link rel="stylesheet" type="text/css" href="css/QuizBox.css">
  • <script defer type="text/javascript" src='example.js'></script>

NOTE: For demonstration purposes the JS file is called example.js. The name of the JS file does not matter.

Step 2:

To create your first quiz, you must first create a QuizBox object in your JS file. This can be done as shown below:

const parentElementOfQuiz = document.querySelector('body');

const quiz = new QuizBox(parentElementOfQuiz);

Now that your have a QuizBox object, you can start adding questions to the quiz. For example, to make a multiple choice question with one correct answer:

quiz.createMultipleChoiceOne("What does HTML stand for?", ["HyperText Machine Learning", "HyperText Markup Language", "Hyper Machine Language", "None of the above", "All of the above"], "HyperText Markup Language");


Other types of questions can be created and added to the quiz in a similar manner.

Below are tables that describe different API functions that can be used to create the Quiz.

Question APIs

Name Description Parameters Return Value Usage

QuizBox

Given the parameter mainDiv, the function will return a JS object that represents a quiz.

mainDiv: is a DOM element that the quiz will appear in the browser.

None
  • Example:

    • Const elementToAddQuizTo = document.querySelector('body');

    • Const newQuiz = new QuizBox(elementToAddQuizTo);

    • NOTE: The quiz will appear in the body element.

createMultipleChoiceOne

Given the parameters, question, options and answer, the function will create a multiple-choice question and add it to the quiz. Users can answer the question by selecting one of the options as their answer to the question.
  • question: is string that represents the question.

  • options: is an array of strings where each string is an answer option to the question.

  • answer: is a string that is contained in the “options” array. This string represents the answer to the question.

None
  • NOTE: If a developer wants to create this type of multiple-choice question, they must first create a QuizBox object.

  • Example:

    • Const elementToAddQuizTo = document.querySelector('body');

    • Const newQuiz = new QuizBox(elementToAddQuizTo);

    • newQuiz.createMultipleChoiceOne (“What is 2 + 2”, [“1”, “2”, “3”, “4”], “4”);

createMultipleChoiceMany

Given the parameters, question, options and answers, the function will create a multiple-choice question and add it to the quiz. Users can answer the question by selecting multiple options as their answer.
  • question: is string that represents the question.

  • options: is an array of strings where each string is an answer option to the question.

  • answer: is an array of strings that are contained in the “options” array. This array of strings represents the answers to the question.

None
  • NOTE: If a developer wants to create this type of multiple-choice question, they must first create a QuizBox object.

  • Example:

    • Const elementToAddQuizTo = document.querySelector('body');

    • Const newQuiz = new QuizBox(elementToAddQuizTo);

    • newQuiz.createMultipleChoiceMany(“Which of the following cities are located in Ontario, Canada?”, [“Hamilton”, “Toronto”, “Mississauga”, “New York”], [“Hamilton”, “Toronto”, “Mississauga”]);

createTrueOrFalseQuestion

Given the parameters, statements and answers, the function will create a true or false question and add it to the quiz. Users can answer the question by dragging tiles (each tile represents a statement) into the red box (statements in this box are considered false) or into the green box (statements in this box are considered true).
  • statements: is an array of strings where each string is a true or false statement.

  • answer: is an array of Boolean values. The value at an index indicates the validity of the statement at the same index in the array “statements”. For example, if the value at index 0 in the array, “answers”, is true then the statement at index 0 in the array, “statements”, is true.

None
  • NOTE: If a developer wants to create a true or false question, they must first create a QuizBox object.

  • Example:

    • Const elementToAddQuizTo = document.querySelector('body');

    • Const newQuiz = new QuizBox(elementToAddQuizTo);

    • newQuiz.createMultipleChoiceMany([“URL stands for Uniform Resource Locator”, “HTTP stands for HyperText Translation Protocol”], [true, false]);

createSequenceQuestion

Given the parameters, events and answer, the function will create a sequence question and add it to the quiz. Users can answer the question by dragging tiles (each tile represents an event) into the blue box chronologically.
  • Events: is an array of strings where each string is an event.

  • answer: is an array of strings that are contained in the array “events”. Note that the event at index 0 is the most recent event and the event at the last index is the oldest event.

None
  • NOTE: If a developer wants to create a sequence question, they must first create a QuizBox object.

  • Example:

    • Const elementToAddQuizTo = document.querySelector('body');

    • Const newQuiz = new QuizBox(elementToAddQuizTo);

    • newQuiz.createSequenceQuestion ([“World War 2”, “World War 1”, “COVID-19”], [“COVID-19”, “World War 1”, “World War 2”]);

createFillInTheBlank

Given the parameters, questions and answers, the function will create a fill-in-the-blank question and add it to the quiz. Users can answer the question by typing into the text boxes.

  • questions: is an array of strings where each string is question. To indicate a blank space, type three underscores without space. For example, Blue is a ___ (three underscores) colour.

  • answer: is an array of strings. The string at each index represents the answer to a blank space in the questions array. Moreover, the string at index 0 represents the answer to the first blank space in questions, the string at index 1 represents the answer to the second blank space in questions, etc.

None
  • NOTE: If a developer wants to create a fill-in-the-blank question, they must first create a QuizBox object.

  • Example:

    • Const elementToAddQuizTo = document.querySelector('body');

    • Const newQuiz = new QuizBox(elementToAddQuizTo);

    • newQuiz.createFillInTheBlank (["Red is a ___ colour.", "RAM stands for ___."], [“primary”, “Random Access Memory”]);

createMatching

Given the parameters, titleOfQuestion, match1, match2 and answers, the function will create a matching question and add it to the quiz. Users can answer the question by clicking a tile from the first column and then clicking a tile from the second column.

  • titleOfQuestion: is a string that represents the title of the question. This title should be a description of what is supposed to be matched.

  • match1: is an array of strings. This array should contain words or phrases that are related to words and phrases in the array, match2.

  • match2: is an array of strings. This array should contain words or phrases that are related to words and phrases in the array, match1.

  • answer: an array of arrays, where the nested array is an array of strings. The strings in the nested array are two words/phrases that are related to each other. Please note that the nested array should contain exactly two strings

None
  • NOTE: If a developer wants to create a Matching question, they must first create a QuizBox object.

  • Example:

    • Const elementToAddQuizTo = document.querySelector('body');

    • Const newQuiz = new QuizBox(elementToAddQuizTo);

    • newQuiz.createMatching ("Matching the following Countries with their captial city", ["Switzerland", "South Korea"], [ "Bern", "Seoul"], [["Switzerland", "Bern"],["South Korea", "Seoul"]]);

getUserAnswers

Returns the quiz's answers submitted by a user. Note that the user's answers will not be recorded and formatted until they submit the quiz. Hence, this function should only be called once it is known that the user has submitted the quiz.

None

Returns an array of JS objects where each JS object contains the user's answers. The format of each JS object depends on the type of question.

Format of each JS Object:
  • Multiple Choice One JS object has the following attributes:

    • questionType: "Multiple Choice one"
    • userAnswer: A string that represents the option that the user picked

  • Multiple Choice Many JS object has the following attributes:

    • questionType: "Multiple Choice Many"
    • userAnswer: An array of strings, where each string represents one of the options that the user picked

  • True Or False JS object has the following attributes:

    • questionType: "True or False"
    • questionsUserAnswerAsTrue: an array of integers where each integer represents a statement number. Statement numbers in this array indicate the statements that the user picked to be true.
    • questionsUserAnswerAsFalse: an array of integers where each integer represents a statement number. Statement numbers in this array indicate the statements that the user picked to be false.
    • NOTE: Statement numbers are determined based on the index the statement was at in the array, statements, which is a parameter that is passed in when creating the true or false question. Please see the "createTrueOrFalseQuestion" section for more information.

  • Sequence JS object has the following attributes:

    • questionType: "Sequence"
    • userAnswer: an array of strings where each string is an event. The order (from left to right) in which the events are in this array, is the order in which the user rerranged the events from most recent to oldest

  • Fill in the blank JS object has the following attributes:

    • questionType: "Fill in the blank"
    • userAnswer: an array of strings. The string at each index represents the user's answer to a blank space. For example, the string at index 0 represents the user's answer to the first blank space, the string at index 1 represents the user's answer to the second blank space, etc.

  • Matching JS object has the following attributes:

    • questionType: "Matching"
    • userAnswer: an array of arrays, where the nested array is an array of strings. The strings in the nested array are the two words/phrases that the user selected to be related to each other.
  • NOTE: If a developer wants to create this type of multiple-choice question, they must first create a QuizBox object.

  • Example:

    • Const elementToAddQuizTo = document.querySelector('body');

    • Const newQuiz = new QuizBox(elementToAddQuizTo);

    • newQuiz.gettingUserAnswers ();

Animation APIs

Name Description Parameters Return Value Usage

setAnimation

Given the parameters animationName and questionNumber, the function will set the transition animation (i.e. animation that occurs when going from one question to another) for a question based on the questionNumber.

Note: setting a transition animation for a question will remove any previous animation that was set for that question.

  • animationName: a string that represents the name of an animation.

    • there are four types of transition animation:

      • Instant.

      • Fade In.

      • Reveal.

      • Slide Down.

      • Scroll.

  • questionNumber: an integer that represents a question number. The animation will be set to the question with this number. For example, if questionNumber is 6, then transition animation will be set to question 6.

  • To see examples of different types of animations click here.

  • returns true if the animation was successfully set to a question

  • otherwise returns false.
  • NOTE: If a developer wants to use this function, they must first create a QuizBox object.

  • Example:

    • Const elementToAddQuizTo = document.querySelector('body');

    • Const newQuiz = new QuizBox(elementToAddQuizTo);

    • newQuiz.createMultipleChoiceOne (“What is 2 + 2”, [“1”, “2”, “3”, “4”], “4”);

    • newQuiz.setAnimation (“Fade In”, 1);

setAnimationAll

Given the parameter animationName, the function will set the transition animation (i.e. animation that occurs when going from one question to another) for all of the questions that are created so far.

Note: setting a transition animation for all of the questions will remove any previous animation that was set.

  • animationName: a string that represents the name of an animation

    • there are four types of transition animation:

      • Instant.

      • Fade In.

      • Reveal.

      • Slide Down.

      • Scroll.

  • To see examples of different types of animations click here.

None
  • NOTE: If a developer wants to use this function, they must first create a QuizBox object.

  • Example:

    • Const elementToAddQuizTo = document.querySelector('body');

    • Const newQuiz = new QuizBox(elementToAddQuizTo);

    • newQuiz.createMultipleChoiceOne (“What is 2 + 2”, [“1”, “2”, “3”, “4”], “4”);

    • newQuiz.setAnimationAll (“Fade In”);
