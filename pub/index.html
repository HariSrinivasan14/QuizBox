What is QuizBox?

QuizBox is a JS library that allows developers to create quizzes easily. The library provides question templates that developers can use to create quizzes. Other features of this library include the ability for developers to choose animations that play when going from one question to another, computing the scores of a quiz once it is completed (given that the developer provides the answers), and the ability for developers to extract user's answers after they complete the quiz.
